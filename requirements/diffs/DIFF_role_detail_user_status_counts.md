# 需求变更：角色详情增加启用/停用用户统计

## 变更标题
角色详情增加启用/停用用户统计字段

## 变更模块
角色管理 - 角色详情

## 当前行为
- 角色详情接口当前返回 `user_count`
- 不区分启用用户数和停用用户数

## 目标行为
- 角色详情接口新增：
  - `active_user_count`
  - `disabled_user_count`
- 保留：
  - `user_count`

## 统计口径
- `active_user_count`：仅统计 `status = 1`
- `disabled_user_count`：仅统计 `status = 2`
- `status = 3`（锁定）与 `status = 4`（未激活）不计入上述两个字段

## 影响范围
- 后端：角色详情接口响应增强
- 前端：可选消费新增字段，旧前端不受影响
- 数据库：无变更
- 权限：无变更
- 测试：补角色详情统计回归测试

## 兼容要求
- 不删除、不重命名现有字段
- 仅追加字段
- 旧前端不改代码也可继续使用

## 验收口径
1. 角色详情接口返回 `active_user_count`
2. 角色详情接口返回 `disabled_user_count`
3. `user_count` 口径保持不变
4. 锁定和未激活用户不计入新增两个统计字段
